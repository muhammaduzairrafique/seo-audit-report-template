# SEO Audit Report Template

A simple SEO audit report structure for reviewing website issues, competitor gaps, keyword opportunities, and priority action steps.

## Report Sections

- Website overview
- Technical SEO issues
- Indexing and crawlability
- On-page SEO review
- Keyword opportunities
- Competitor analysis
- Content gaps
- Backlink and authority notes
- Core Web Vitals notes
- Priority action plan

## Purpose

This template helps organize SEO findings into a clear report that shows what is wrong, why it matters, and what should be fixed first.
